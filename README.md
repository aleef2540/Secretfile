# Secretfile

# 📂 DocuVault - Internal Document Management System

A robust, full-stack web application designed for organizations to securely store, organize, and manage internal documents. This project showcases a modern **React** frontend powered by **Vite** and a high-performance **C# .NET** backend.

https://secretfile-9952.web.app/

---

## 🚀 Key Features

* **Full CRUD Operations:** Seamlessly Create, Read, Update, and Delete document records.
* **Centralized Storage:** A unified dashboard to view and manage all organizational files.
* **Search & Filter:** Quickly locate documents using optimized search queries.
* **Lightweight Database:** Powered by SQLite for fast, zero-configuration data management.
* **Responsive UI:** Built with React and Vite for a lightning-fast user experience.
